## Project Requirement
Build an open source module for generating a unique formatted sequence. For example, an invoice module requires an invoice number to have a format like "ss/cc/yy/xxxx", where ss should be replaced by state code, cc should be replaced by country code, yy will be replaced by current financial year and xxxx will be replaced by a unique sequence number. The module should be flexible to handle different replacer string. Also the system should be able to handle large request concurrently without creating any duplicate sequence or 2 requests getting same sequence number. 


## Solution
Create a sequence number generator service for which the client will register a template of sequence number describing the replacer for particular string. Store these data in a db. When a request is made by the client by passing the template id, the service will perform related replacements and generate a unique number attach it to the sequence number post replacing the template and return the response. Store the counter value in redis against the key generated by replacing the placeholders in the template. Using lua script check and update counter value should be atomic. 

## Tech Stack
-Java and Spring boot for backend service with redis integration for counters
-Create a UI using react and typescript for registering the template and also AI assisted dialogues for getting the replacer for fields in the template
